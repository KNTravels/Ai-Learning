## 🚀 How to Use

### 1. Local Development
```bash
# Option 1: Direct browser open
open index.html

# Option 2: Python server
python3 -m http.server 8000

# Option 3: Node.js
npm run serve
```

### 2. Add New Topics
```bash
# 1. Edit data/topics.json
# 2. Run generator
npm run generate

# 3. Update sitemap.xml
```

### 3. Deploy
```bash
# GitHub Pages
git push origin main

# Netlify
# Connect repository → Auto deploy

# Vercel
# Import → Auto deploy
```
